ProfitLoss (C#)
===============================================================================

Overview
-------------------------------------------------------------------------------

This example demonstrates using the TT API to retrieve the P&L (profit & loss) 
for a single instrument.  


Instructions
-------------------------------------------------------------------------------

1. Drag and drop an Instrument from the Market Grid in X_TRADER.
2. View P&L updates.
3. Select the different calculation and formatting options.


TT API Objects
-------------------------------------------------------------------------------

InstrumentTradeSubscription
